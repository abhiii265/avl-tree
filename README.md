# Animated AVL Tree Project

## Overview
This project provides two implementations of an AVL Tree:

1. **Web-based Visualization**:
   - An interactive AVL Tree visualizer built with HTML, CSS, and JavaScript.
   - Allows users to insert and clear nodes, with animations displayed on a canvas.

2. **Command-line Implementation**:
   - A Java-based AVL Tree implementation that accepts input from the user and prints the tree structure.

---

## Files

### Web-based Visualization
- **`index.html`**: Entry point for the web application.
- **`style.css`**: Styles for the web interface.
- **`avl.js`**: JavaScript logic for the AVL Tree and animations.

### Command-line Implementation
- **`AVLTree.java`**: Java implementation of the AVL Tree.

---

## How to Use

### Web-based Visualization
1. Open `index.html` in any modern browser.
2. Use the input box to enter numbers and click "Insert" to add nodes.
3. Click "Clear" to reset the tree.

### Command-line Implementation
1. Compile the Java file:
   ```
   javac AVLTree.java
   ```
2. Run the program:
   ```
   java Main
   ```
3. Follow the prompts to input numbers and view the tree structure.

---

## Features
- **AVL Tree Properties**: Maintains balance after each insertion.
- **Interactive Visualization**: Animations for node insertion and tree balancing.
- **Command-line Interface**: Prints the tree structure in a readable format.

---

## Requirements
- **Web-based Visualization**: A modern browser (e.g., Chrome, Firefox).
- **Command-line Implementation**: Java Development Kit (JDK).
